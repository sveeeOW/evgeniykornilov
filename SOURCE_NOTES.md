# Старый лендинг Евгения Корнилова — исходники

Это сохранённая старая версия лендинга с WOW-стилистикой, Telegram-чатами/сообщениями, фото-слайдером и движущейся лентой.

## Что внутри
- `src/App.tsx` — основной React-компонент лендинга
- `src/index.css` — стили Tailwind / кастомная CSS-обвязка
- `src/main.tsx` — точка входа
- `public/images/` — исходные изображения для hero и слайдеров
- `package.json` — зависимости и команды проекта
- `vite.config.ts`, `tailwind.config.js`, `postcss.config.cjs`, `tsconfig*.json` — конфиги сборки

## Как запустить локально
```bash
npm install
npm run dev
```

## Как собрать для продакшена
```bash
npm run build
```
После сборки готовая статическая версия появится в папке `dist/`.
